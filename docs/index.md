# 欢迎来到竺院辅学计划网站

😉你好~

这里是竺可桢学院学业指导中心辅学计划的站点。辅学计划为竺院大一新生提供数学分析（H）、线性代数（H）、微积分（H）、普通物理学（H）、普通化学（H）、程序设计基础等**荣誉课程**的学习指导与帮助。
