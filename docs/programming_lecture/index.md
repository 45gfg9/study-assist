## 课程简介

程序设计与算法基础课程讲述的内容局限于语言本身，较为表象。实质上，C 语言是一门很难的编程语言，不懂编译原理、操作系统和计算机体系结构无法获得深入的理解，而这些也是学习计算机专业的同学应当尽早了解的知识。我们希望通过线下辅学授课，为具有 C 语言基础或愿意深入学习计算机基础知识的同学提供更深入、更底层方面的知识扩展，让同学们在计算机系统而非算法题的语境下理解 C 语言。
